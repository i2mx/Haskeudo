# Haskeudo
A Cambridge Pseudo Code compiler written in Haskell

## Usage
`hsc <sourcefile>.hsc <outputLocation1>.cpp <outputLocation2>.cpp`

https://github.com/i2mx/Haskeudo/blob/main/assets/example.mp4
